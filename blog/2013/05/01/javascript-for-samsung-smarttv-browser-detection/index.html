JavaScript for Samsung SmartTV Browser Detection

Here’s the code to determine if the current browser is a SmartTV - requires Modernizr: [code language=”javascript”] function isSmartTV() { return Modernizr.websockets === false; } [/code] This works because the Samsung SmartTV does not support Web Sockets which I find out by running the Modernizr Test Suite on the SmartTV. UPDATE: Actually I wound up changing this to use User-Agent sniffing based on a great discussion I had with Derek Anderson [code language=”javascript”] function () { return navigator.userAgent.search(/TV/i) >= 0; } [/code]
